# 쿠러기 (khlugy)

경희대학교 중앙 IT 동아리 쿠러그(KHLUG) 멤버가 만든 서비스를 외부에 공개·호스팅하는 플랫폼입니다.  

멤버는 서비스를 등록 신청하고, 운영진 검토를 거쳐 쿠러기에 게시됩니다.  

추가로 서브도메인 호스팅을 신청한 서비스는 `*.khlugy.app`으로 호스팅될 수 있습니다.  

그 외에도 프런트엔드/백엔드 호스팅 등을 신청할 수 있습니다.  

## 주요 기능

- 서비스 목록 확인 및 소개 페이지
- 서비스 등록 신청 (서비스명 중복 확인, 카테고리, 그룹 및 참여자 선택, 호스팅 옵션, 서브도메인 신청)
- 소개 문구 마크다운 지원
- 운영진 신청 목록 조회 및 상태 관리

## 페이지 라우팅

- `/` — 서비스 목록(전체)
- `/submit` — 서비스 등록 신청 (멤버)
- `/manage/submissions` — 신청 목록 및 검토 (운영진)
